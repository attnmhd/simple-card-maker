# Card Maker

Simple card maker

Case study sample

## Installation

Clone Project

```bash
  git clone https://github.com/attnmhd/simple-card-maker.git
```

Install packages

```bash
  composer install
  npm install
```

Rename env.example -> env & generate APP_KEY

```bash
  php artisan key:generate
```

Run project

```bash
  php artisan serve
  npm run dev
```
